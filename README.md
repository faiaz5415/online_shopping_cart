# 🛒 Online Shopping Cart System

A Dart-based command-line shopping cart simulation. The system allows users to browse items, add them to the cart, and view a final bill with quantity and total price.

## ✨ Features
- Displays available products with prices
- Accepts user input to add items to the cart
- Manages product quantity if item is added multiple times
- Calculates total price at checkout
- Uses basic functions, list operations, and control flow

## 📦 Technologies
- Dart
- List & Map
- Functions
- Looping & if-else control flow

## ▶️ How to Run
```bash
dart shopping_cart.dart
